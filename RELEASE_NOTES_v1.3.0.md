# v1.3.0 - Incident Management

## Overview

This release adds incident management on top of the alerting workflow. Critical alerts can now create incidents that can be assigned, acknowledged, resolved, closed, and audited through a full incident history timeline.

## Features

- Added incident lifecycle management
- Added critical alert to incident creation flow
- Added incident assignment
- Added incident acknowledgement
- Added incident resolution with notes
- Added incident closure
- Added incident history timeline
- Added incident dashboard page
- Added API Gateway routing for incident APIs
